# v0.3.0 — Cloud workspaces

Tier 3: Ship Cloud managed infrastructure. Remote agent execution, team collaboration, real-time coordination.

## Theme

A team working on the same project gets cloud workspaces — isolated containers with the repo checked out, agent running, output streamed live to the UI. This is the primary revenue tier.

## Features

- Cloud workspaces — Rivet Sandbox + agent execution

## Acceptance Criteria

- [ ] Cloud workspace provisions in under 60 seconds
- [ ] Agent output streams to UI in real-time via WebSocket
- [ ] Multiple team members see live workspace status
- [ ] Containers isolated per workspace, auto-terminate on idle
